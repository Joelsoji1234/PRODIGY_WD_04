Projects

OLMS

As part of a comprehensive Database Management Systems (DBMS) project, our team developed an Online Learning Management System (OLMS) platform utilizing React for the front-end and AWS for the back-end infrastructure. This platform was designed to streamline the process of course management, offering features such as user authentication, course creation, and content delivery. Leveraging React's component-based architecture allowed for a dynamic and responsive user interface, enhancing the overall user experience. AWS services, including Amazon RDS for database management and AWS Lambda for serverless computing, provided a robust and scalable back-end solution. This project not only demonstrated my ability to integrate modern web development frameworks with cloud services but also reinforced my skills in database design, system architecture, and full-stack development.
